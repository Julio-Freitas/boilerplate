# 🏫 This is a boilerplate using:
 - nextjs
 - typescript
 - styled-component
 - next-pwa
 -jest
 -react-testing-library

## Getting Started
clone this project
  - git clone https://github.com/Julio-Freitas/boilerplate.git

## 💻 In your terminal

```bash
cd boilerplate && npm install
# or
cd boilerplate  yarn
```
## 👟 Running 
```bash
npm run dev
# or
yarn dev
```
